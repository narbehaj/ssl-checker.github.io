SSL Checker

A simple Python script that collects SSL/TLS information from hosts.

GitHub PyPI

Overview

SSL Checker connects to one or more hosts and reports on their SSL/TLS certificates: issuer, validity dates, days until expiry, TLS version, hostname matching and more. It supports certificate chain validation, weak-configuration detection, STARTTLS, parallel scanning, monitoring exit codes, and export to JSON, CSV, or HTML.

Install

pip install python-ssl-checker

Or run it directly from a clone of the repository:

git clone https://github.com/narbehaj/ssl-checker.git
cd ssl-checker
pip install -r requirements.txt

Usage

# Check one or more hosts
./ssl_checker.py -H example.com github.com:443

# Run a security analysis
./ssl_checker.py -H narbeh.org:443 -a

# Probe which TLS versions are accepted
./ssl_checker.py -H github.com -p

# Inspect a mail server over STARTTLS
./ssl_checker.py -H smtp.gmail.com:587 -T smtp

# Scan a file of hosts in parallel, monitoring mode
./ssl_checker.py -f hosts.txt -n 20 -w 30 -q -e

# JSON output
./ssl_checker.py -j -H narbeh.org:443

Example output

[✓] github.com

Options

FlagDescription
-H, --hostHosts to check, separated by space
-f, --host-fileRead hosts from a file (use - for stdin)
-s, --socksConnect through a SOCKS proxy (host:port)
-c, --csvExport results to a CSV file
-j, --jsonPrint results as JSON
-S, --summaryShow summary output only
-x, --htmlExport results to an HTML file
-J, --json-saveSave JSON to a file per host
-t, --timeoutConnection timeout in seconds (default: 5)
-r, --retriesRetries on a dropped connection (default: 2)
-a, --analyzeRun SSL security analysis
-p, --protocolsProbe acceptance of TLS 1.0–1.3
-T, --starttlsUse STARTTLS (smtp, imap, pop3, ftp, xmpp)
--ca-fileCustom CA bundle for verification
-n, --concurrencyNumber of parallel host checks (default: 1)
-w, --warning-daysExpiry warning threshold in days (default: 15)
-q, --quietOnly print problematic hosts
-e, --exit-codeNagios-style exit codes
-v, --verboseVerbose troubleshooting output
-h, --helpShow the help message
